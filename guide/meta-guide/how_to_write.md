### How to Write a Tree Structure in Markdown


| Symbol | Unicode              | Use                               |
| ------ | -------------------- | --------------------------------- |
| `│`    | U+2502               | Vertical line (used for branches) |
| `├──`  | U+251C U+2500 U+2500 | Intermediate item                 |
| `└──`  | U+2514 U+2500 U+2500 | Last item in branch               |
